# Authman

#### 介绍

使用spring boot开发的通用权限管理系统。

#### 软件架构

软件架构说明

> 后台框架：Spring Boot 2.0、MyBatisPlus

> 权限：Shiro、Ehcache

> 前端：Layui、Thymeleaf

> 数据库：MySQL5.7

#### 安装教程

1. 解压文件，用idea作为maven工程打开
2. 创建数据库
   创建数据库：authman  编码：UTF-8
   执行SQL语句（authman.sql）
3. 修改application.yml中数据库密码
4. 启动项目
5. 访问系统：localhost:8081/authman，用户名;admin 密码:123

#### 功能说明

1. 部门管理
2. 权限管理
3. 角色管理
4. 用户管理
5. 日志管理
6. 系统登录和注销
7. 修改密码
